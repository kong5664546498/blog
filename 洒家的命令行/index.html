忽略别名

$ \cat # 忽略叫 "cat" 的别名 - 具体解释: https://stackoverflow.com/a/16506263/22617
$ command cat # 忽略函数和别名

使用的插件

plugins=(git zsh-autosuggestions zsh-syntax-highlighting command-not-found z)

语法高亮, 自动提示, 再来一个 z 命令解放 cd

经常用的就这三个了
