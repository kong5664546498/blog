2018-10-25 / 2 min read

RPM 方式部署

  • 下载 java_rpm

  • 查看该系统是否安装过Java 环境，如果安装过将其卸载安装最新的版本，更换Java 的版本也可以这样去操作
    rpm -qa | grep jdk

  • 如果已安装有其它版本 则 卸载 列出多少个版本插件就要卸载几个插件 完成后再次检查是否有遗漏
    rpm -e --nodeps java-1.7.0-openjdk-headless-1.7.0.111-2.6.7.2.el7_2.x86_64

  • 将下载后的文件上传到目标环境相应路径下 在该文件下进行安装
    rpm -ivh jdk-8u131-linux-x64.rpm

  • 检验安装
    java -version

  • 查看安装目录

    which java   
ls -lrt /usr/bin/java
ls -lrt /etc/alternatives/java

  • 配置环境变量

    • vim etc/profile

      JAVA_HOME=/usr/java/jdk1.8.0_151
JRE_HOME=/usr/java/jdk1.8.0_151/jre
CLASSPATH=$JAVA_HOME/lib:$JRE_HOME/lib:$CLASSPATH
PATH=$JAVA_HOME/bin:$JRE_HOME/bin:$PATH

  • 加载环境变量
    source /etc/profile

  • 验证环境变量

    echo $JAVA_HOME 
echo $CLASSPATH 
echo $PATH

tar.gz 安装

  • 下载 官链 百度云

  • 配置环境变量

    • vim .zshrc
      zsh 真好用 没有使用 zsh 的 修改 .bashrc

      JAVA_HOME=/usr/local/java/jdk1.8.0_181
JRE_HOME=$JAVA_HOME/jre
JAVA_BIN=$JAVA_HOME/bin
CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar:$JRE_HOME/lib
PATH=$PATH:$JAVA_HOME/bin:$JRE_HOME/bin
export JAVA_HOME JRE_HOME PATH CLASSPATH

apt 安装

sudo apt-get install openjdk-8-jdk
